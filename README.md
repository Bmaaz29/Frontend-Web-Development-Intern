# Zomato Card UI Clone 🍽️
A responsive Zomato-style restaurant card UI built using HTML and CSS.  
This project replicates the restaurant card design similar to the Zomato food platform.

## 📷 Project Preview
![Zomato Card UI](image/cards.png)

## 🚀 Features
- Zomato-style restaurant cards
- Offer badge (30% OFF)
- Restaurant rating badge
- Food category display
- Responsive design for mobile, tablet, and desktop
- Clean and modern UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- Responsive Design
